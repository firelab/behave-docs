Torching Tree Height

behaveplus:crown:input:spotting:torching_trees:torching_tree_height:torching_tree_height:help

Torching Tree Height is the height measured from the ground to the top of a torching tree. For group tree torching, it is the average height of the torching trees. Torching Tree Height can be different than Downwind Canopy Height.

Torching Tree Height is Used to Calculate

  • Maximum Spotting Distance from Torching Trees
