Wind Driven Surface Fuel Model Code

behaveplus:surface:input:fuel_models:standard:wind-driven-fuel-model:wind-driven-fuel-model:help

This spotting model was developed for grass fuel types, so a subset of fuel models are available to be selected. This spotting model can only be used for a wind-driven fire in surface fuels that are not sheltered from the wind by an overstory.
