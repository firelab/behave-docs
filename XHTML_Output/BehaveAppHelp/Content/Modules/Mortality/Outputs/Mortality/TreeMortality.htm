Scorch Height

behaveplus:mortality:input:scorch:fire:scorch_height:scorch_height:help behaveplus:mortality:output:scorch:scorch_height:scorch_height:help behaveplus:mortality:output:tree_mortality:tree_mortality:scorch-height:help

Scorch Height is the maximum vertical height above the ground at which lethal scorching of crown foliage occurs. Below this height, the temperature is above the lethal temperature necessary to kill live foliage. This temperature is assumed to be 140 degrees F (60 degrees C).

Scorch Height is an automated output in the Surface/Mortality module.

Tree Crown Length Scorched

behaveplus:mortality:output:tree_mortality:tree_mortality:tree_crown_length_scorched:help

Tree Crown Length Scorched is the vertical length of the crown that is scorched.

Total Crown Length = tree height x crown ratio

Tree Crown Length Scorched = scorch height - (tree height - total crown length)

Percent Crown Length Scorched = 100 x (crown length scorched/total crown length)

The following species use the tree crown length scorched model.

Scientific Name Common Name
Abies magnifica red fir
Abies concolor white fir
Calocedrus decurrens incense cedar
Pinus lambertiana sugar pine

Refer to (https://www.frames.gov/behave/documentation-tips-history) for additional documentation about tree mortality in the Key Documents.

Tree Crown Volume Scorched

behaveplus:mortality:output:tree_mortality:tree_mortality:tree_crown_volume_scorched:help

Tree Crown Volume Scorched is the percentage of the tree crown volume that is scorched.

Total Crown Length = tree height x crown ratio

Crown Length Scorched = scorch height - (tree height - total crown length)

Percent Crown Volume Scorched = 100 x (crown length scorched x (2 x total crown length - crown length scorched)) / total crown length ^2

The following species use the tree crown volume scorched model.

Scientific Name Common Name

Mortality Equation

behaveplus:mortality:input:fuelvegetation_overstory:mortality-equation:mortality-equation:help behaveplus:mortality:output:tree_mortality:tree_mortality:mortality-equation:help

Crown Scorch and Bark Char are the two groups of mortality equations used in the Surface/Mortality module. The mortality equation used depends on the selected tree species. Behave automatically selects the equation in the Surface/Mortality module based on the best available science.

The bark char (also referred to as bole char in other publications) equations are specific to eastern hardwood species and the crown scorch equations apply to all other available species.

Refer to (https://www.frames.gov/behave/documentation-tips-history) for additional documentation about tree mortality in the Key Documents.

Bark Thickness

behaveplus:mortality:output:tree_mortality:tree_mortality:bark_thickness:help

Bark Thickness is used to calculate tree mortality for some species. Bark thickness is assumed to have a linear relationship to diameter at breast height (DBH).

Bark thickness = v x DBH, where v is a species-specific bark thickness coefficient.

Refer to (https://www.frames.gov/behave/documentation-tips-history) for additional documentation about tree mortality in the Key Documents.

Bole Char Height

behaveplus:mortality:input:fuelvegetation_overstory:bole-char-height:bole-char-height:help behaveplus:mortality:output:tree_mortality:tree_mortality:bole_char_height:help

Bark Char Height is measured from the ground to the highest point on the bole (main stem) of a tree that char is detected. This output is automatically provided when one of the species below is selected.

Bark Char equations are considered the best available science to predict the probability of mortality for ten hardwood species. The equations predict mortality using species, diameter at breast height (DBH), and bark char height.

This set of equations was developed by Keyser et al. (2018) from 210 plots on 13 National Parks in the Southern U.S. The data were collected typically two years following late dormant season prescribed burns. Bark char height was measured from the ground to the highest point of char regardless of slope position.

If one of the following species is selected in the Surface/Mortality module, Behave will automatically select the bark char equation.

Scientific Name Common Name(s)
Acer rubrum red maple
Cornus florida flowering dogwood
Nyssa sylvatica blackgum, tupelo, black tupelo
Oxydendrum arboreum sourwood
Quercus alba white oak
Quercus coccinea scarlet oak, red oak
Quercus marilandica blackjack oak
Quercus montana chestnut oak, tanbark oak
Quercus velutina black oak
Sassafras albidum sassafras, white sassafras

Refer to (https://www.frames.gov/behave/documentation-tips-history) for additional documentation about tree mortality in the Key Documents.
