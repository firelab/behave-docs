Mortality

behaveplus:mortality:help

This module provides the predicted probability of tree mortality based on fire behavior outputs from the Surface module. Due to this module interaction, the intent is to evaluate the thresholds for tree mortality based on fire behavior, such as flame length. Tree mortality is a common objective in prescribed fire plans.

Crown scorch and bark char are the two groups of mortality equations used in the Surface/Mortality module. The mortality equation used depends on the selected tree species. Behave automatically selects the equation in the Surface/Mortality module based on the best available science and displays this information in the Run Results.

  • Crown Scorch – Each species is assigned a mortality model and a bark thickness equation.

  • Bark Char (also referred to as bole char in other publications) – Equations are specific to eastern hardwood species.

Refer to the Behave Documentation for additional documentation about tree mortality in the Key Documents.
