Canopy Height

Canopy Height is the average height of the dominant and co-dominant trees in a stand.

The downwind canopy height is an input required to calculate the maximum spotting distance from torching trees.

Canopy Height is Used to Calculate

  • Maximum Spotting Distance from an Active Crown Fire

  • Tree Crown Length Scorched (Probability of Mortality)
