Line Construction Offset

behaveplus:contain:input:suppression:line_construction_offset:line_construction_offset:help

Line Construction Offset is the distance from the fire to where the suppression line is constructed. A construction offset of zero indicates direct attack. Any other line construction offset value indicates an indirect attack.

When running CONTAIN, using a line construction offset depends on the predicted flame lengths and the type of resources used. For example, if predicted flame lengths are 5-6 ft and you are using hand crews as the resources in CONTAIN, you should use a line construction offset greater than zero.
