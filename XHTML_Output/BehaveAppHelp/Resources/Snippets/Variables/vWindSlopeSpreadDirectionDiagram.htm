Wind/Slope/Spread Direction Diagram

behaveplus:surface:output:fire_behavior:surface_fire:windslopespread-direction-diagram:help

This diagram is an automated output that will be displayed in the Run Results if a user selects either of the following:

  • Direction of Interest as the direction mode rather than Heading or Heading, Backing, Flanking

  • Wind and slope are not aligned.

The diagram displays the direction of heading (maximum spread), flanking, backing, and wind. It also displays the direction for the direction of interest if used. The y-axis (the axis that goes up) is considered uphill.
