Resource Name

behaveplus:contain:input:suppression:resources:resource_name:help

Resource Name is a one-word identifier for each containment resource. The resource name must be a single word without any blanks or tabs. However, hyphens and underscores can be used. A few examples are as follows:

  • Engine1

  • IHC_1

  • T2_Crew_2

  • Dozer-1
