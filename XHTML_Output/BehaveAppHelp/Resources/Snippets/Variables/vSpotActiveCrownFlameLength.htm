The Active Crown Fire Flame Length is added as an input when you select Maximum Spotting Distance from an Active Crown Fire as an output and have not selected any fire behavior behavior outputs, such as rate of spread or flame length.

This input can be based on observed crown fire flame lengths or an estimated crown fire flame length.
