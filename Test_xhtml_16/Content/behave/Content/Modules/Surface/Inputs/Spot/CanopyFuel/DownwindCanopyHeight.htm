Downwind Canopy Height (SURFACE)

Downwind Canopy Height is the average height of the canopy downwind from torching trees. Downwind Canopy Height can be different than Torching Tree Height.

Downwind Canopy Height is Used to Calculate
  • Maximum Spotting Distance from Torching Trees
