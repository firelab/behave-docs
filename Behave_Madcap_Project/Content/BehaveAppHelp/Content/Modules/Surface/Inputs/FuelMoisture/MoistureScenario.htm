Moisture Scenario (SURFACE)

A moisture scenario is a set of values representing the moisture content of each category of surface fuel (see Fuel Moisture). A fuel moisture scenario is similar to a fuel model – it is a pre-packaged set of values used to complete runs. With a fuel moisture scenario, all dead and live fuel moisture values are combined into a single input, thus making outputs easier to understand.

The moisture scenarios used to develop the 40 fuel models are included with Behave (Scott and Burgan 2005). Users can also create fuel moisture scenarios relevant to local areas and weather conditions, or for developing elements within a prescribed fire plan.
